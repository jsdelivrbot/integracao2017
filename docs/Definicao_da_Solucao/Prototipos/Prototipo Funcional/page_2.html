Arquiteturas

Atualmente existem vários modelos de bancos de dados tais como orientado a objetos, orientado a documentos, etc. Mas o mais comum ainda é o banco de dados relacional. A decisão entre qual modelo de banco de dados utilizar baseia-se no tipo de dados que você pretende armazenar. Por exemplo, se você for armazenar uma grande quantidade de dados em um modelo pequeno (imagine um banco de dados pro twitter), é mais indicado utilizar um banco de dados orientado a documentos a um banco de dados relacional. Muitas questões envolvem essa decisão, mas não é uma questão de superioridade entre uma ou outra tecnologia, todas tem prós e contras e são mais indicadas ou não para cada problema.
